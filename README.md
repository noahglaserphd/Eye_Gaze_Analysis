# Video-Based Gaze Analysis from Eye-Tracking Overlays

An open-source research pipeline for extracting and analyzing gaze behavior from video recordings that contain a visible gaze overlay.

The system detects gaze markers embedded in recorded videos and converts them into structured gaze data such as gaze coordinates, fixations, and saccades. These data can then be analyzed through statistical summaries, spatial heatmaps, scanpaths, and an interactive dashboard.

This pipeline was developed primarily for human–computer interaction (HCI), usability testing, and gameplay research, but it can be used in any situation where gaze overlays appear in recorded video.

Unlike traditional eye-tracking analysis tools that depend on proprietary ecosystems, this approach works directly with screen recordings containing a gaze cursor or overlay, allowing analysis without access to vendor APIs or specialized analysis software.

---

# Project Workflow

The project consists of two stages.

### Offline Processing Scripts

These scripts extract gaze data from videos and compute gaze metrics.

### Interactive Dashboard

The dashboard allows researchers to explore sessions, inspect time windows, view visualizations, and export results.

The dashboard does **not process raw videos**. It visualizes data that has already been generated by the processing scripts.

# Installation

Python **3.9 or newer** is recommended.

## Create a Conda Environment

```bash
conda create -n gaze_dash python=3.10
conda activate gaze_dash
```

## Install Required Libraries

```bash
pip install streamlit plotly pandas numpy opencv-python matplotlib
```

## Install FFmpeg (required for video previews)

```bash
conda install -c conda-forge ffmpeg
```

Optional (for exporting Plotly figures as PNG):

```bash
pip install kaleido
```

---

# Step 1 — Prepare Videos

Place recordings containing gaze overlays inside the `videos/` folder.

Example:

```
videos/
    Rec1.mkv
    Rec2.mkv
    Rec3.mp4
```

Each filename becomes the **session identifier** used throughout the analysis.

---

# Step 2 — Extract Gaze Coordinates From Video

Run the gaze extraction script:

```bash
python extract_dot.py
```

This script scans all videos inside the `videos/` directory and detects the gaze overlay position in each frame.

The script automatically generates several output folders if they do not already exist:

```
gaze_samples/
fixations/
saccades/
metrics/
```

Outputs include:

- gaze coordinate time series  
- detected fixation events  
- detected saccade events  
- summary fixation statistics  

This step performs the **core processing of the pipeline**.

---

# Step 3 — Generate Session Visualizations

```bash
python make_figures.py
```

This produces figures for each session including:

- fixation heatmaps  
- scanpath diagrams  
- fixation duration distributions  
- saccade amplitude distributions  

Figures are saved in a `figures/` directory.

---

# Step 4 — Generate Time Window Metrics

```bash
python window_analysis.py
```

This script divides each session into **30-second windows** and calculates gaze metrics for each window.

Output is written to:

```
time_windows/
```

---

# Step 5 — Generate Aggregate Figures

```bash
python make_aggregate_figure.py
```

This creates summary figures across all sessions.

These figures allow comparison of gaze behavior across participants or recordings.

---

# Step 6 — Launch the Interactive Dashboard

Start the dashboard:

```bash
streamlit run app.py
```

The dashboard will open automatically in your browser.

---

# Dashboard Capabilities

The dashboard allows researchers to:

- select recorded sessions  
- inspect **30-second time windows**  
- view fixation heatmaps  
- visualize scanpaths  
- examine fixation duration distributions  
- analyze saccade amplitude distributions  
- view synchronized video clips  
- export charts and video segments  
- export full-session outputs  

The dashboard is designed for **exploratory analysis and qualitative inspection** of gaze behavior.

---

# Time Window Analysis

Each session is divided into **30-second windows**.

Example segmentation:

```
0–30 seconds
30–60 seconds
60–90 seconds
```

For each window the system calculates:

- fixation count  
- mean fixation duration  
- total fixation time  
- mean saccade amplitude  

These metrics allow researchers to examine **temporal changes in visual attention** during a task.

---

# Optional Scripts

Some scripts in the repository serve specialized purposes.

**analyze_gaze_csv.py**  
Used when you already have gaze coordinate CSV files and want to compute fixations and saccades without running the video extraction step.

**summarize_fixations.py**  
Computes summary statistics from fixation CSV files.

**inspect_fixations.py**  
Provides a quick visual inspection tool for fixation data.

These scripts are **not required for the standard video-based pipeline**.

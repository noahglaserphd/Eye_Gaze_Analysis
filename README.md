# Video-Based Gaze Analysis from Eye-Tracking Overlays

An open-source pipeline for extracting and analyzing gaze behavior from **video recordings that contain a visible gaze overlay**.

This project converts gaze markers embedded in recorded videos into structured gaze data such as gaze coordinates, fixations, and saccades. These data can then be analyzed through statistical summaries, spatial heatmaps, scanpaths, and an interactive dashboard.

The system is designed primarily for **human–computer interaction (HCI), usability testing, and gameplay research**, but it can be applied to any scenario where gaze overlays appear in recorded video.

Unlike traditional eye-tracking analysis tools that rely on proprietary ecosystems, this pipeline works directly with **screen recordings that contain a gaze cursor or overlay**, allowing analysis without direct access to device APIs.

---

# Project Workflow

The workflow consists of two stages.

1. **Offline processing scripts**

   These scripts extract gaze coordinates from videos and compute derived gaze metrics such as fixations, saccades, and time-window summaries.

2. **Interactive dashboard**

   The dashboard allows researchers to explore sessions, inspect time windows, review visualizations, and export analysis results.

The dashboard **does not process raw videos**. It visualizes data that has already been generated by the processing scripts.

# Installation

## 1. Install Python

Python **3.9 or newer** is recommended.

---

## 2. Create a Conda Environment

```
conda create -n gaze_dash python=3.10
conda activate gaze_dash
```

---

## 3. Install Dependencies

```
pip install streamlit plotly pandas numpy opencv-python matplotlib
conda install -c conda-forge ffmpeg
```

Optional dependency (required for exporting figures as PNG):

```
pip install kaleido
```

---

# Step 1 — Prepare Videos

Place recordings containing gaze overlays inside the `videos/` folder.

Example:

```
videos/
   Rec1.mkv
   Rec2.mkv
   Rec3.mp4
```

The filename becomes the **session identifier** used throughout the pipeline.

---

# Step 2 — Extract Gaze Coordinates

Run the video processing script.

```
python process_videos.py
```

This step detects the gaze marker in each frame and produces the following outputs:

```
gaze_samples/
fixations/
saccades/
```

These files contain the extracted gaze coordinates and derived gaze events.

---

# Step 3 — Generate Summary Metrics

Run the fixation summarization script.

```
python summarize_fixations.py
```

This step produces summary statistics describing fixation behavior.

---

# Step 4 — Generate Static Figures

Generate visualization outputs.

```
python make_figures.py
```

This script creates figures such as:

- fixation heatmaps
- scanpaths
- fixation duration distributions
- saccade amplitude distributions

Outputs are saved to:

```
figures/
```

---

# Step 5 — Generate Time Window Metrics

Divide sessions into fixed time windows and compute window-based metrics.

```
python window_analysis.py
```

Each session is divided into **30-second windows**, and metrics are computed for each window.

Output files are written to:

```
time_windows/
```

These files are used by the interactive dashboard.

---

# Step 6 — Launch the Interactive Dashboard

Start the dashboard:

```
streamlit run app.py
```

The dashboard provides interactive exploration of gaze behavior.

---

# Dashboard Capabilities

The dashboard allows researchers to:

- select recorded sessions
- inspect **30-second time windows**
- view fixation heatmaps
- visualize scanpaths
- examine fixation duration distributions
- analyze saccade amplitude distributions
- view synchronized video clips
- export charts and video segments
- export full-session analysis outputs

The dashboard is intended for **exploratory analysis and qualitative inspection** of gaze behavior.

---

# Time Window Analysis

Each recording is divided into fixed **30-second windows**.

Example segmentation:

```
0–30 seconds
30–60 seconds
60–90 seconds
```

For each window the system calculates:

- fixation count
- mean fixation duration
- total fixation time
- mean saccade amplitude

These metrics help researchers identify **temporal shifts in visual attention and interaction behavior**.
